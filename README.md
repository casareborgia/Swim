## 💡 "내 수영 영상 한 편으로 받는 정밀 AI 코칭 리포트"

수영 영상을 스마트폰으로 찍어 올리면, AI가 프레임 단위로 수영 동작을 추적하고 자세 결함을 정밀 분석해 드립니다. 단순 점수 매기기 방식이 아닌, **시각적 증거(타임스탬프 또는 키프레임 번호)에 기반한 팩트 위주의 진단**과 **우선순위 기반의 맞춤형 교정 드릴**을 처방하여 이메일로 받아볼 수 있는 똑똑한 AI 수영 파트너입니다.

---

## ✨ 핵심 기능 (Features)

* **3회 교차 분석 (Self-Consistency Verification)**: 동일한 영상을 3회 독립적으로 반복 분석하여 일관되게 검출된 진짜 결함만 채택합니다. AI의 무작위 거짓 양성(환각) 진단을 방지하여 극도의 신뢰도를 확보합니다.
* **촬영 각도 적합성 필터링 (Angle Gating)**: 카메라의 촬영 각도(측면, 정면, 후면)에 따라 판정 가능한 항목을 사전에 필터링합니다. 각도에 맞지 않는 항목은 억지로 판정하지 않고 "판정 불가(not_visible)"로 처리해 정확성을 극대화합니다.
* **증거 중심의 코칭 리포트**: "캐치 타이밍 지연", "하체 처짐" 등의 모든 결함에 대해 영상 속 정확한 타임스탬프와 정성적 분석 원인을 제공합니다.
* **4대 영법 통합 진단**: 자유형 · 배영 · 평영 · 접영의 4영법을 완벽히 지원하며, 영법별 다각도 전용 스키마를 탑재했습니다.
* **운영 검수 파이프라인**: 인공지능이 임시 작성한 리포트 초안을 관리자 검수 대기열(`/admin`)에서 실제 수영 코치가 눈으로 재검수한 뒤, 안전하게 이메일로 발송하는 베타 운영 모델을 갖추고 있습니다.

## 🛠 아키텍처 및 시스템 흐름

```
[React SPA] 수영 영상 및 정보 접수 폼 → POST /api/submit
                                ↓
[FastAPI]   접수 데이터 수신 → 백그라운드 태스크 기동 (Gemini API 3회 병렬 분석 실행)
                                ↓
[FastAPI]   분석 결과 집계 및 다수결 기반 리포트 초안 생성 → 검수 대기열 등록
                                ↓
[운영자/코치] 검수 웹 화면(/admin)에서 진단 내용 최종 승인 → 사용자에게 이메일로 HTML 리포트 발송
```

### 디렉토리 구성
* **`server.py`**: FastAPI 백엔드 (업로드 처리, 시간당 IP당 어뷰징 방지 레이트 리미트, 검수 대기열 API 및 어드민 페이지 제공)
* **`report_render.py`**: 정밀 분석 JSON 결과를 유저가 보기 편한 수려한 HTML/Markdown 리포트로 렌더링
* **`swim-web/`**: 사용자를 매료시키는 Modern UI 프론트엔드 (Vite + React 19 + TypeScript + Tailwind CSS 4)

## ⚙️ 실행 방법 및 설정

> ⚠️ **중요 (비공개 자산에 대한 안내)**
> 수영 코칭의 정수인 영법 판정 스키마 및 마스터 프롬프트 템플릿(`private_assets/` 폴더 하위)은 카피캣 방지 및 핵심 자산 보호를 위해 공개 저장소에서 제외되어 있습니다. 
> 로컬 실행 시 `SWIM_ASSETS_DIR` 환경변수로 스키마 파일들이 포함된 경로를 제공해야 정상 작동합니다.

### 1) 의존성 설치
```bash
# 백엔드 의존성 설치 (+ 시스템에 ffmpeg 설치가 필요합니다)
pip install -r requirements.txt

# 프론트엔드 빌드
npm install --prefix swim-web
npm run build --prefix swim-web
```

### 2) 환경 변수 (.env) 설정
프로젝트 루트에 `.env` 파일을 생성하고 아래 항목들을 정의합니다:
```env
GEMINI_API_KEY=your_gemini_api_key   # Gemini API 키 (코드 하드코딩 금지)
ADMIN_TOKEN=your_admin_secret_token  # /admin 검수 페이지 접근 비밀 토큰
SWIM_ASSETS_DIR=./private_assets     # 프롬프트 MD 파일 및 스키마 경로
MAX_UPLOAD_MB=50                     # 최대 비디오 업로드 용량
FREE_CREDITS=1                       # 이메일당 무료 제공 횟수
RATE_LIMIT_PER_HOUR=10               # IP당 시간당 최대 분석 요청 횟수
```

### 3) 로컬 서버 구동
```bash
python3 -m uvicorn server:app --host <HOST> --port 8010
```
